AI Agent 一键接入

推荐直接跑本地一键路演脚本（Windows / PowerShell）：

powershell -ExecutionPolicy Bypass -File "E:\\claude code\\agent-auto-make\\office\\demo-agent.ps1"

脚本会自动执行：join → status(plan/search/code/test/deploy) → report → present。

如果你要手动接入，再用下面这个命令拿 token：

curl -s -X POST http://localhost:19000/api/join \
  -H "Content-Type: application/json" \
  -d '{"name":"我的龙虾","role":"AI Agent"}'

拿到 token 后，调用 /api/status、/api/report、/api/meeting/present 即可完成路演。
